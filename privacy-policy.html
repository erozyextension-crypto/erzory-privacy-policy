Erzory

Privacy Policy

Last updated: May 19, 2026

Applies to Chrome extension version 1.0.0.

Erzory is a manual spelling and grammar correction extension for Spanish and English. Its only purpose is to review text written by the user in editable fields and show correction suggestions.

1. Information processed

  • Text written by the user in supported editable fields such as input, textarea, and some contenteditable editors.
  • Website content only to the extent needed to locate and review the active editable field.
  • Local extension settings such as selected language, enabled state, and ignored words.
  • Feedback data voluntarily submitted by the user through the feedback form: name, email address, and opinion message.

2. How the information is used

  • To detect spelling and grammar issues.
  • To generate local correction suggestions.
  • To request contextual suggestions from LanguageTool.
  • To save local extension preferences.
  • To send feedback messages submitted by the user.

Erzory does not use processed data for advertising, profiling, resale, or unrelated analytics.

3. Third-party services

Erzory shares limited data with third-party services only when required by the feature being used.

LanguageTool

  • Purpose: grammar and context suggestions
  • Data sent: reviewed text and selected language
  • Endpoint: https://api.languagetool.org/*

EmailJS

  • Purpose: deliver user feedback to the developer
  • Data sent: name, email address, and opinion message entered in the feedback form
  • Endpoint: https://api.emailjs.com/*

PayPal

  • Purpose: optional support payment
  • Data sent by the extension: none beyond opening the PayPal link

4. Local storage and retention

  • Extension settings are stored locally in the browser with Chrome storage.
  • Local settings remain until the user changes them, clears extension storage, or uninstalls the extension.
  • The extension does not intentionally store reviewed text on a proprietary backend.
  • Third-party services may process or retain data according to their own policies.

5. User controls

  • Pause or disable revision from the popup.
  • Uninstall the extension at any time.
  • Avoid using the feedback form if they do not want to send personal contact data.

6. Security

Erzory sends third-party requests only over HTTPS.
